# **App Name**: AquaLeads CRM

## Core Features:

- Dashboard Overview: Interactive dashboard displaying booking reports, yacht lists, and revenue summaries (daily, monthly, yearly).
- Comprehensive Lead Tracking: Detailed lead management with custom fields (ID, Agent, Status, Month, Yacht, etc.), including financial data and commission calculations. Allows for data entry and updates for all specified fields related to leads.
- Role-Based User Access: User role management to define permissions and designations within the CRM.
- CSV Data Management: Import and export of data in CSV format for easy data management and integration with other systems.
- Customizable Admin Settings: Admin settings panel for user management, theme customization (color and layout), and logo uploading.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) to reflect the nautical theme and trustworthiness.
- Background color: Light blue (#D5DBDB) for a clean, spacious feel.
- Accent color: Yellow-Orange (#E67E22) to highlight key actions and information.
- Clean, modern, sans-serif fonts for optimal readability across all devices and screen sizes.
- Nautical-themed icons for quick visual navigation.
- Clean and structured layout with clear sectioning to promote data visibility and usability.
